## 2. Node.jsのインストール

### Windowsユーザー

[Node.jsの公式サイト](https://nodejs.org/ja/download)にアクセスします。
`Windows`が指定されていることを確認して、赤枠の**Windowsインストーラー**をクリックしてダウンロードします。
![](https://github.com/user-attachments/assets/79fd10be-483e-4fde-aed6-762dc2587111)

https://zenn.dev/kuuki/articles/windows-nodejs-install

ターミナルを開いて`brew`がインストールされているか確認します。


既にbrewがインストールされています。以下の記事 **[③ Node.jsのインストール](https://zenn.dev/nakohama/articles/911374261e8b33#%E2%91%A2-node.js%E3%81%AE%E3%82%A4%E3%83%B3%E3%82%B9%E3%83%88%E3%83%BC%E3%83%AB)** からはじめてください。

## 3. GitHubからリポジトリのクローンを行う

任意のディレクトリ(フォルダ)に移動して、ターミナルを開きます。

12月の勉強会では、HTTPS接続を行ってクローンを行いましたので、ターミナルに以下のコマンドを入力します。

```sh
git clone https://github.com/miry41/haji-web.git
```

HTTPS接続ではなく、SSH接続を行いたい場合は以下のコマンドを入力してください。

```sh
git clone git@github.com:miry41/haji-web.git
```

`haji-web`ディレクトリ(フォルダ)がお手元のPC内にあるか確認してください。

VSCodeを開いたあと、`haji-web`ディレクトリをVSCodeにドラッグ&ドロップすることでプロジェクトを開くことができます。

---

## 4. npmでパッケージをインストールする

プロジェクトを開いた状態で、VSCodeでターミナルを開きます。
(Windowsなら`Ctrl + Shift + @`でターミナルが出現します。)

ターミナルに以下を入力してください。

```sh
npm install
```

これだけで、実行に必要なJavaScriptパッケージを準備することができます。

## 5. check.shの実行

### Windowsユーザー

Windowsでは、pwsh(PowerShell)でこのファイルを実行することができません。代わりに、**Git Bash**を使用します。こちらは、Gitのインストール時に同梱されているので追加の導入は不要です。

ターミナルの左側に下向きの矢印があるので、こちらをクリックして**Git Bash**をクリックします。

![](https://github.com/user-attachments/assets/e4ad8b01-2820-4ded-8345-36de3046ddbd)

Git Bashが表示されたら、以下のコマンドを入力します。

```sh
./check.sh
```

実行結果が表示されます。問題なく導入できているものにはチェックマークがつきます。
![](https://github.com/user-attachments/assets/6cb61e60-18cd-49de-830a-63c25e82143a)

**ALL CHECKS PASSED (6 Passed)** が表示されていれば環境構築が完了です。

### macOSユーザー

VSCodeのターミナルに、以下のコマンドを入力します。

```sh
./check.sh
```

#### Permission Deniedと表示される場合

実行権限がないので、権限を付与する必要があります。VSCodeのターミナルに、以下のコマンドを入力します。

```sh
chmod +x ./check.sh
```

このコマンドを入力したら、再度`./check.sh`を実行してみてください。

実行結果が表示されます。問題なく導入できているものにはチェックマークがつきます。
![](https://github.com/user-attachments/assets/d940b5ed-93cc-4f76-bb83-e2484ebda0e0)

**ALL CHECKS PASSED (6 Passed)** が表示されていれば環境構築が完了です。
